

Install `figlet`. (Mac):

```bash|{type: 'command', platform: 'darwin'}
brew install figlet
```

(Windows):

```bash|{type: 'command', privileged: true, platform: 'win32'}
choco install figlet-go -y
```

`figlet` will translate the given text into an ascii banner. Try it out!

```bash|{type: 'command'}
figlet docable
```
