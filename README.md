# Docable Notebooks

Create a VM to host the server and execute the notebook:

```bash
bakerx run
```

### Try a PoC demo

Open http://192.168.33.33:3000/ in your browser, click on "browse", upload [`./resources/steps.yml`](./resources/steps.yml), and click on "Run Notebook" button.
